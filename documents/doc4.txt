Information retrieval is the area behind search engines, digital libraries, and document search tools. The main problem is simple to describe: a user has an information need, writes a query, and the system must return useful documents. The difficult part is deciding which documents are relevant. Information retrieval systems usually preprocess documents, build indexes, process queries, and rank results. An inverted index is especially important because it helps the system find documents quickly without scanning every file from the beginning each time. This idea is small enough to implement in a class project but powerful enough to appear in real search systems.
