Evaluation metrics help explain whether a retrieval system is performing well. Precision measures how many retrieved documents are actually relevant. Recall measures how many relevant documents were successfully retrieved. F1-score combines precision and recall into one balanced value. To calculate these metrics, the developer needs a small set of test queries and a manually prepared ground truth list of relevant documents. Evaluation is useful in the report because it moves the project beyond implementation and shows a comparison between Boolean retrieval, phrase search, and ranked retrieval.
