Wildcard and partial matching help users search when they remember only part of a word. A prefix match treats the input like query star, so learn can match learning. A suffix match treats it like star query, so ing can match mining or processing. A contains match treats it like star query star, so the input can appear anywhere inside the term. This feature works well with the vocabulary from the inverted index. It is also easy to demonstrate because the interface can show which terms matched and which documents were retrieved from those terms.
