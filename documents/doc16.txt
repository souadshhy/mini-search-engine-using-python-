Python is popular in education because its syntax is readable and it has many useful libraries. In an information retrieval project, Python can read text files, clean strings, count words, build dictionaries, and calculate similarity scores. Libraries such as pandas can help display tables, while Streamlit can turn a script into a simple web interface. Even when libraries are used, the main retrieval logic should be understood clearly. Building an inverted index, a positional index, and TF-IDF ranking by hand is a good way to learn how search engines work internally.
