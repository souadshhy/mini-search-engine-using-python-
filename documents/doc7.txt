Boolean retrieval uses logical operators such as AND, OR, and NOT to combine search terms. A query like data AND mining returns documents that contain both terms. A query like python OR java returns documents that contain at least one of the terms. A query using NOT removes documents containing a specific term. Boolean retrieval is easy to understand and useful when the user knows exactly what they want. Its weakness is that it does not naturally rank documents. A document either matches the condition or it does not, so additional ranking methods are needed for more flexible searching.
